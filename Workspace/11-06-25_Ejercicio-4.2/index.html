Fútbol

Sistema de puntos

Se otorgan 3 puntos por victoria, 1 punto por empate y 0 puntos por derrota.

El objetivo principal para ganar es marcar más goles que el equipo contrario durante el tiempo reglamentario.

Reglas básicas

Cada equipo juega con 11 jugadores en el campo, incluyendo un portero.
El balón debe cruzar completamente la línea de gol para que se considere un gol.
No se permite usar las manos para controlar el balón, salvo el portero dentro de su área.
